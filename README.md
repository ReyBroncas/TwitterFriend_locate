# TwitterFriend locator


It is a simple python web app which uses Twitter API to locate user specifed friends.


### Installation

Install the dependencies and run app.py to test it.

```sh
$ pip install folium
$ pip install oauth2
$ pip install geopy
$ pip install flask
```
